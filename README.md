## 🚀 Getting Started

1. **Clone the repository**
	 ```powershell
	 git clone <your-repo-url>
	 ```
2. **Navigate to the Practice folder**
	 ```powershell
	 cd "E2 Ruby programming/Practice"
	 ```
3. **Run any Ruby file**
	 ```powershell
	 ruby filename.rb
	 ```
